Reverse Integer

Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range [-231, 231 - 1], then return 0.

Assume the environment does not allow you to store 64-bit integers (signed or unsigned).

Example 1:

Input: x = 123
Output: 321

Example 2:

Input: x = -123
Output: -321

Example 3:

Input: x = 120
Output: 21

Example 4:

Input: x = 0
Output: 0

Constraints:

  • -231 <= x <= 231 - 1

class Solution {
     public int reverse(int x) {
         int sign;
         if(x == 0){
             System.out.println(x);
         }else if(x > 0){
             sign = 1;
         }else if(x < 0){
             sign = -1;
             x = - x;
         }
         int result = 0;
         for(int i = x;i < 100;i = i / 10){//循环条件错误
             result = result * 10 + i % 10 * 10;//不该还乘以10
         }
         if(sign == 1){
             System.out.println(result / 10);
         }else if(sign == -1){
             System.out.println(- result / 10);
         }//无return，不符合提交规则         
     }
}

//@author Blankj
class Solution {
    public int reverse(int x) {
        long res = 0;
		for (; x != 0; x /= 10)
			res = res * 10 + x % 10;
		return res > Integer.MAX_VALUE || res < Integer.MIN_VALUE ? 0: (int) res;
    }
}

==知识点==

  • Integer.MAX_VALUE指取最大值，为正最大值；Integer.MIN_VALUE指取最小值，为负最大值

  • Integer.MAX_VALUE+1=Integer.MIN_VALUE

  • 因为：

    Integer.MAX_VALUE的二进制是0111 1111 1111 1111 1111 1111 1111 1111 Integer.MIN_VALUE的二进制是 1000 0000 0000 0000 0000 0000 0000 0000

  • || ——短路或

    a||b，a为false时，b执行，a为true时，b不执行，但|、||运算结果都相同

  • 取余的余数永远是正的

class Solution {
     public int reverse(int x) {

         long result = 0;
         for(int i = x;i != 0; i /= 10){//判断条件
             result = result * 10 + i % 10;//除以10
         }
         if(x > Integer.MAX_VALUE || x < Integer.MIN_VALUE){//条件大于小于0
             result = 0;
         }
         if(x < Integer.MAX_VALUE || x > Integer.MIN_VALUE){
             result = (int)result;
         }
         return (int)result;//强制转换，提交规则return
     }
}
